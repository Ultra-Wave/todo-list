# To-Do List App

A simple and responsive to-do list application built using HTML, CSS, and JavaScript.  
This project allows users to manage daily tasks efficiently with data stored locally in the browser.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent data using localStorage
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
